Enunciado

Marcos, um técnico de manutenção, está registrando a quantidade de vezes que cada máquina em uma fábrica apresenta falhas. Ele quer saber a soma das falhas em máquinas com números ímpares para verificar se há um padrão de falhas nessas máquinas.

Escreva uma função que recebe uma lista de números e retorne a soma dos elementos ímpares da lista.

Entrada: A entrada consiste em uma lista de números inteiros. A lista pode ser vazia ou ter até 100 elementos.

Saída: A função deve retornar um número inteiro representando a soma dos elementos ímpares da lista.

{% endif %}
